# Recreate Image With Voronoi

Aplicação de Algoritmos Genéticos:
 1. Basicamente o algoritmo particiona o espaço e sobre cada região faz a média das cores, então pinta a região com essa média.
 2. O Algoritmo Genético busca a melhor posição para os pontos dilatarem para gerar uma imagem com o menor desvio padrão em relação a imagem original. 
 3. O fitness é a variância.
